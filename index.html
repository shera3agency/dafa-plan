الشهر الأول: الأساسيات والبنية التحتية

بناء البنية التحتية

🔍 SEO — التأسيس

  • Keyword Research: منظفات سعودية، صابون ملابس، شامبو عباية
  • Technical Audit: سرعة الموقع، هيكل URL، Sitemap
  • تحسين On-page لصفحات المنتجات الأكثر مبيعاً
  • إضافة Schema Markup وربط Google Search Console & Analytics

🎨 واجهة المتجر (UX)

  • تحسين الصفحة الرئيسية مع USP واضح
  • إضافة Badge "منتج سعودي 100%"
  • إضافة Trust Signals: شهادات الجودة ومدة التوصيل
  • تحسين Core Web Vitals وصور المنتجات

📱 TikTok Ads

  • إعداد TikTok Business + Pixel
  • إنتاج 6 فيديوهات UGC (قبل / بعد)
  • حملة Awareness ميزانية 500–1000 ريال
  • استهداف: نساء 25-45، مهتمين بالمنزل والتنظيف

🌐 Meta Ads

  • إعداد Pixel + Conversion API
  • إنشاء Catalog وحملة Awareness فيديو
  • A/B Test: فيديو vs صورة

الشهر الثاني: التوسع والتفعيل

التوسع والانتشار

📝 SEO Content

  • نشر 4 مقالات شهرياً
  • وصف منتجات متوافق مع SEO
  • بدء استراتيجية Link Building
  • تحسين صفحات الفئات الرئيسية

🛒 تطوير المتجر

  • تفعيل عروض الباقات (Bundle Offers)
  • تفعيل Cross-sell / Upsell
  • تحسين صفحة Checkout وزر واتساب
  • إنشاء Landing pages مخصصة للحملات

📱 TikTok

  • زيادة الميزانية للحملات الناجحة
  • إطلاق حملات Conversion

  • إطلاق تحدي (Challenge) هاشتاج
